LetterGenerator
==============
This application generates letters for intermediaries, TSPs, and banks.

Requirements:
- Windows 10 or later
- Microsoft Visual C++ Redistributable (download from https://aka.ms/vs/17/release/vc_redist.x64.exe if prompted)

Usage:
- Double-click LetterGenerator.exe to run.
- Generated letters are saved in Documents\GeneratedLetters\{inter|tsp|bank}.
- Logs are saved in Documents\LetterGeneratorLogs\app.log for troubleshooting.

Troubleshooting:
- If antivirus flags the .exe, add an exception for LetterGenerator.exe.
- If errors occur, check the log file and contact the developer.
